Политика конфиденциальности

Последнее обновление: 2026-01-26

Разработчик: Николай Явтушенко

Приложение: «Умножение» (тренажёр таблицы умножения)

Кратко

Это приложение не собирает и не обрабатывает персональные данные пользователей. Приложение использует ограниченную анонимную агрегированную статистику (например, общее количество запусков приложения и решённых учебных примеров за день), которая не позволяет идентифицировать пользователя и не связана с конкретным устройством или ребёнком.

Какие данные обрабатываются

  • Учебные данные на устройстве: набор решённых примеров, ошибки, затраченное время, локальные настройки (язык/тема). Эти данные используются только для работы «специального режима» с повторением самых сложных для вас примеров.
  • Анонимная агрегированная статистика использования: общее количество запусков приложения и решённых примеров за день. Эти данные используются исключительно для понимания общей нагрузки и улучшения качества приложения. Статистика не содержит идентификаторов пользователей, устройств или сессий.

Мы не обрабатываем: имя, адрес электронной почты, номер телефона, геолокацию, контакты, фото/видео, рекламные идентификаторы и т. п.

Сторонние сервисы

  • Аналитика: используется только собственная серверная аналитика в агрегированном и анонимном виде. Приложение не использует сторонние аналитические SDK.
  • Реклама: отсутствует.
  • Передача третьим лицам: не осуществляется.

Детская конфиденциальность (Kids)

Приложение предназначено для детей и соответствует требованиям Apple для категории Kids. Используемая агрегированная статистика не является персональными данными, не используется для отслеживания поведения отдельных пользователей и не позволяет идентифицировать ребёнка. Приложение не содержит рекламы, трекинга, пользовательского контента, чатов или ссылок на внешние ресурсы.

Разрешения устройства

Приложение не запрашивает доступ к камере, микрофону, контактам, фото, геолокации и другим чувствительным разрешениям.

Безопасность данных

Все данные хранятся локально в памяти приложения/на устройстве. Вы можете удалить их, удалив приложение.

Ваши права

Поскольку персональные данные не собираются и не обрабатываются на наших серверах, запросы на доступ/удаление/переносимые копии не применимы. Если у вас есть вопросы касательно конфиденциальности, свяжитесь с нами.

Изменения политики

Мы можем обновлять эту Политику при добавлении новых функций (например, аналитики или рекламы). В таком случае документ будет обновлён, а дата изменения — актуализирована. Для детских приложений любые новые сторонние сервисы будут внедряться только в соответствии с требованиями Apple и действующим законодательством.

Используемые шрифты и иконки

В приложении используется шрифт Borsok для отображения текста интерфейса.

Privacy Policy

Last updated: 2026-01-26

Developer: Nikolay Yavtushenko

App: “Multiplication” (times table trainer)

Summary

This app does not collect or process personal data. It uses limited anonymous aggregated usage statistics (such as the total number of app launches and solved exercises per day) which cannot identify users and are not linked to any individual child or device.

Data Processed

  • On-device learning data: solved items, mistakes, time spent, local settings (language/theme). This is used solely to power the “special mode” that repeats the most difficult items for you.
  • Anonymous aggregated usage statistics: total number of app launches and solved exercises per day. This data is used solely to understand overall usage and improve the app. No user identifiers, device identifiers, or session tracking are collected.

We do not process: name, email, phone number, location, contacts, photos/videos, advertising identifiers, etc.

Third Parties

  • Analytics: limited first-party analytics in an aggregated and anonymous form. No third-party analytics SDKs are used.
  • Advertising: none.
  • Data sharing: none.

Children’s Privacy (Kids)

The app is intended for children and complies with Apple’s Kids category requirements. Any aggregated usage statistics collected are non-personal, do not enable tracking of individual users, and cannot be used to identify a child. The app contains no advertising, tracking, user-generated content, chats, or external links.

Device Permissions

The app does not request access to camera, microphone, contacts, photos, location, or other sensitive permissions.

Data Security

All data is stored locally on the device/in the app’s storage. You can delete it by removing the app.

Your Rights

Because no personal data is collected or processed on our servers, requests for access/deletion/portability are not applicable. For any privacy questions, please contact us.

Changes to This Policy

We may update this Policy when adding new features (e.g., analytics or ads). In such cases, we will update this document and the effective date. For kids apps, any new third-party services will be implemented only in compliance with Apple’s requirements and applicable laws.

Fonts and Icons

The application uses the Borsok font for displaying the user interface text.

The application also uses graphical icons from the Basic UI Game Icons Set (Community), available on Figma: https://www.figma.com/design/e8uvRD9gh9vTyJ4EfrsWO6/Basic-Ui-Game-Icons-Set–Community-?m=auto
